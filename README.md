# Hunter
#### BASH-based username availability checker. Fast, efficient, and compatible with a variety of platforms. Hunter will assist you in spotting unique aliases.

------------

##### List of websites.
- **YouTube**
- **SoundCloud**
- **Reddit**
- **Github**
